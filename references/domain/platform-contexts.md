# 平台语境与信号质量

## 为什么需要区分平台语境

同样的文本放在不同的平台语境中，信号价值和含义截然不同。

- B站粉丝在评论区玩梗 → 可能是社区文化，不反映品牌认知
- 小红书的推荐笔记 → 有策划成分，不是纯消费者感受
- 淘宝评论区 → 最接近真实消费者信号

**不区分平台语境 = 信号质量污染。**

## 六种平台语境

### 1. 粉丝场

| 属性 | 说明 |
|------|------|
| 典型平台 | B站官方号评论区、微博超话、品牌粉丝群 |
| 信号特征 | 滤镜强、正面偏高、存在"圈内梗"和社区黑话 |
| 信号等权 | **不得等同大众消费者心智** |
| 典型误导 | 粉丝说"永远支持你" ≠ 大众消费者感受 |
| 处理原则 | 标注 signal_owner=community，单独统计不合并 |

### 2. 公众讨论场

| 属性 | 说明 |
|------|------|
| 典型平台 | 小红书（非商业笔记）、知乎问答、微博广场、豆瓣小组 |
| 信号特征 | 表达相对真实，有正有负，但可能被热门帖/算法偏差扭曲 |
| 信号价值 | 🟡 中高 — 最接近公众真实感受 |
| 处理原则 | 注意区分"热门话题"和"真实消费体验" |

### 3. 官方表达场

| 属性 | 说明 |
|------|------|
| 典型平台 | 品牌官方号、官网、官方新闻稿 |
| 信号特征 | 品牌希望的自我表达，完全策划 |
| 信号价值 | 🔴 低 — 仅反映品牌意图，不反映消费者感受 |
| 处理原则 | 仅用于对比品牌表达 vs 消费者感知，不作为消费者洞察 |

### 4. 达人种草场

| 属性 | 说明 |
|------|------|
| 典型平台 | 小红书达人笔记、抖音带货视频、公众号推广 |
| 信号特征 | 有策划成分（可能受品牌付费影响），半真实 |
| 信号价值 | 🔴🟡 中低 — 有参考价值但需标注"有策划成分" |
| 处理原则 | 仅在达人评论文本表现出个性化表达时才视为真实信号 |

### 5. 搜索行为

| 属性 | 说明 |
|------|------|
| 典型场景 | 搜索下拉词、相关搜索、搜索指数趋势 |
| 信号特征 | 反映**未满足的需求**，是消费者主动表达的信号 |
| 信号价值 | 🟡🟢 中高 — 需求信号纯净但缺少情感全貌|
| 处理原则 | 作为消费者需求的辅助验证，不单独作为原型判定依据 |

### 6. 真实消费评论

| 属性 | 说明 |
|------|------|
| 典型平台 | 淘宝/京东评论区、大众点评、应用商店评价 |
| 信号特征 | 最真实、未过滤，包含正负两面真实的消费体验 |
| 信号价值 | 🟢 高 — 最高置信度的消费者信号 |
| 处理原则 | 权重最高，是原型判定的"黄金标准" |

## 平台上下文标签

每条数据在编码时必须标注 `platform_context` 字段：

| 标签 | 对应场景 |
|------|---------|
| fan_field | 粉丝场（B站官号、微博超话） |
| public_discussion | 公众讨论（小红书广场、知乎、微博广场） |
| official_expression | 官方表达（品牌号、官方新闻） |
| influencer_planting | 达人种草（付费/非付费达人的内容评论） |
| search_behavior | 搜索行为（搜索词、下拉词） |
| authentic_review | 真实消费评论（电商评价、点评） |

## 混合语境处理

当一个分析包含多种语境时：

1. **不得合并统计**。不同语境的数据分开编码、分开统计。
2. **报告必须说明**"以下结论中，XX 数据来自粉丝场，不代表大众心智"。
3. **如果某原型信号主要来自粉丝场**，在结论中标注"该信号可能存在社区滤镜"。


## 语境转换

同一个平台的内容可能属于不同语境，取决于具体数据来源：

| 平台 | 可能语境 | 判断依据 |
|------|---------|---------|
| B站 | fan_field / public_discussion | 是官方号评论区还是路人讨论区 |
| 小红书 | public_discussion / influencer_planting / official_expression | 是素人帖、达人合作帖还是品牌号 |
| 微博 | fan_field / public_discussion | 超话还是广场 |
| 淘宝 | authentic_review | 默认真实消费评论 |

## 常见错误

| 错误 | 后果 | 防范 |
|------|------|------|
| B站粉丝评论当消费者心智 | 品牌形象过于正面 | 标注 fan_field，单独分析 |
| 达人投放评论当真实反馈 | 信号偏向正面且有策划偏差 | 标注 influencer_planting，降权处理 |
| 把搜索词当情感完整描述 | 只看到需求看不到感受 | 搜索词仅作为辅助验证 |
