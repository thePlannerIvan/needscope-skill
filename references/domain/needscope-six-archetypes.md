# NeedScope 六原型定义与边界

## 核心原则

六个原型没有优劣之分。它描述的是品牌在消费者心智中占据的情感区域，**不是品牌的"性格测试得分"**。一个品牌低分不一定说明问题——如果策略就是单一原型深耕，其他原型信号少是正常的。

## 六原型定义

### 突破型 (Red)

| 维度 | 说明 |
|------|------|
| **消费者买的是** | "赢"——对抗、征服、打破常规的快感 |
| **正向表达（品牌拥有）** | 突破、超越、不甘平庸、挑战极限、打破规则、反叛、自由、冒险、征服、统治 |
| **负向表达（品牌缺乏）** | 保守、守旧、平庸、畏缩、循规蹈矩、毫无野心、无趣的安全 |
| **品类常见关联** | 运动鞋（竞速/PB）、能量饮料、越野车、电竞外设、功能饮料 |
| **与身份型的边界** | 突破型是"我击败了挑战"，身份型是"我比你高一档"。前者对内（自我超越），后者对外（压制他人）。|
| **与愉悦型的边界** | 突破型有对抗性和目标（征服、打破），愉悦型是一种释放和享受（快乐、好玩）。 |

### 身份型 (Purple)

| 维度 | 说明 |
|------|------|
| **消费者买的是** | "我比你高一档"——地位、稀缺、掌控感 |
| **正向表达（品牌拥有）** | 高贵、独特、稀缺、经典、尊贵、精英、品味、成功、权力、专属 |
| **负向表达（品牌缺乏）** | 廉价、大众、普通、随处可见、没有排面、不够档次 |
| **品类常见关联** | 奢侈品、高端汽车、商务服饰、高端护肤（贵妇线）、威士忌 |
| **与掌控型的边界** | 身份型是"这和我身份匹配"，掌控型是"我做了最聪明的选择"。前者重社会地位，后者重个人专业判断。 |
| **与突破型的边界** | 见突破型。 |

### 掌控型 (Blue)

| 维度 | 说明 |
|------|------|
| **消费者买的是** | "我做了最聪明的选择"——理性、专业、透明 |
| **正向表达（品牌拥有）** | 专业、精准、科技、严谨、数据、可靠、安全、可信、权威、认证、无添/无香/成分党 |
| **负向表达（品牌缺乏）** | 不专业、虚假宣传、智商税、不稳定、不可靠、不信任、不透明 |
| **品类常见关联** | 护肤成分党、运动装备（科技面护）、汽车（操控/安全）、数码评测、金融服务 |
| **与身份型的边界** | 见身份型。 |

### 呵护型 (Brown)

| 维度 | 说明 |
|------|------|
| **消费者买的是** | "对自己好一点"——舒适、自然、被照顾 |
| **正向表达（品牌拥有）** | 柔软、舒适、温和、滋养、包容、治愈、宁静、自然、无负担、温暖、拥抱、放松、安心 |
| **负向表达（品牌缺乏）** | 刺激、粗糙、生硬、冷漠、不舒服、不够温和、冰冷 |
| **品类常见关联** | 护肤品（舒缓线）、家居服、母婴、酒店（度假型）、茶/草药饮品 |
| **与归属型的边界** | 呵护型是"被温柔对待"，归属型是"和大家一样很安心"。前者是被照顾，后者是融入群体。 |

### 归属型 (Orange)

| 维度 | 说明 |
|------|------|
| **消费者买的是** | "不出错、和大家一样"——日常、百搭、实用 |
| **正向表达（品牌拥有）** | 百搭、经典、日常、实用、基础、必备、每个人、公认、不出错、稳、耐穿 |
| **负向表达（品牌缺乏）** | 不接地气、挑人、难搭配、太个性/太奇怪、不实用、不日常 |
| **品类常见关联** | 基础款服饰、大众护肤品、日用消费品、便利店食品、通勤包 |
| **与呵护型的边界** | 见呵护型。归属型的安心感来自"不出错"，呵护型的安心感来自"被拥抱"。 |

### 愉悦型 (Yellow)

| 维度 | 说明 |
|------|------|
| **消费者买的是** | "当下的快乐"——有趣、自由、松弛、好看 |
| **正向表达（品牌拥有）** | 好看、好看/可爱、有趣、好玩、快乐、自由、轻松、愉悦、惊喜、治愈漫/可爱/萌、赏心悦目 |
| **负向表达（品牌缺乏）** | 无聊、沉闷、不好看、老气、死板、过于严肃 |
| **品类常见关联** | 彩妆、甜品、文创、快时尚、社交游戏、周边/联名 |
| **与突破型的边界** | 见突破型。愉悦型的快乐是"好玩"，突破型的快乐是"赢了"。 |

## 边界与组合

### 常见张力组合
- **掌控 × 愉悦**：专业但不枯燥，严谨但有设计感（如一些设计师科技品牌）
- **呵护 × 突破**：柔和但有力量，温柔但不柔弱
- **身份 × 归属**：想要独特但又不想太出格（如轻奢品牌）

### 通常不兼容的组合
- **身份 × 归属**：一个追求稀缺，一个追求普及，很难同时成立
- **突破 × 呵护**：一个对抗世界，一个拥抱世界，消费者诉求相反

### 特殊品类适配
- 同一个词在不同品类指向不同原型："科技"在运动鞋 → 掌控型；在汽车 → 身份型
- "设计感"在服装 → 身份型或愉悦型；在家电 → 掌控型
- 不要硬套通用词典，根据品类语境判断

## 编码注意事项

1. **sentiment 是"对象拥有/缺乏该特质"**，不是评论情绪的正面负面。
   - "这个牌子一点都不松弛" → 归属型 negative
   - "它太温和了不适合我" → 呵护型 negative（消费者认为品牌过度呵护）
2. **一条文本可以有多个原型**（但不是必须）。
3. **"好"和"不好"不直接映射到某个原型**。需要看上下文判断具体心理需求。
4. **文化差异**：中文中"松弛感"可能指向归属型（不出错、不费力）或愉悦型（自由、放松），取决于上下文。
